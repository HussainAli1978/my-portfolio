# Hussain Ali Portfolio Website

A modern, responsive one-page portfolio website for Hussain Ali — Senior Managing Consultant and Microsoft Dynamics 365 F&O Solution Architect.

## Features

- Modern 2026 visual style with glassmorphism, bento cards and gradient accents
- Fully responsive layout for mobile, tablet and desktop
- Accessible semantic HTML5 with skip link, focus states, ARIA labels and reduced-motion support
- SEO metadata including Open Graph, Twitter Cards and Schema.org JSON-LD
- Dark/light mode with localStorage preference persistence
- Responsive hamburger navigation
- Download CV button
- Working email, WhatsApp and LinkedIn contact links
- Separate CSS and JavaScript files for maintainability

## Customization Before Publishing

Replace `your-github-username` in `index.html`, `robots.txt`, and `sitemap.xml` with your real GitHub username/repository URL.

## Deployment

Upload this folder to a GitHub repository and enable GitHub Pages from repository settings.
